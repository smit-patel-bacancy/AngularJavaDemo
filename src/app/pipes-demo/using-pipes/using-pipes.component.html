Using Built-in Angular Pipes

📅 Date Pipes

DatePipe

Original: {{ currentDate }}
date: {{ currentDate | date }}
date:'short': {{ currentDate | date:'short' }}
date:'medium': {{ currentDate | date:'medium' }}
date:'long': {{ currentDate | date:'long' }}
date:'full': {{ currentDate | date:'full' }}
date:'MM/dd/yyyy': {{ currentDate | date:'MM/dd/yyyy' }}

🔤 Text Pipes

UpperCasePipe & LowerCasePipe

Original: {{ sampleText }}
uppercase: {{ sampleText | uppercase }}
lowercase: {{ sampleText | lowercase }}

TitleCasePipe

Original: {{ sampleText }}
titlecase: {{ sampleText | titlecase }}

💰 Number Pipes

CurrencyPipe

Original: {{ price }}
currency: {{ price | currency }}
currency:'EUR': {{ price | currency:'EUR' }}
currency:'USD':'symbol':'1.2-2': {{ price | currency:'USD':'symbol':'1.2-2' }}

DecimalPipe

Original: {{ decimalNumber }}
number:'1.2-2': {{ decimalNumber | number:'1.2-2' }}
number:'1.4-4': {{ decimalNumber | number:'1.4-4' }}

PercentPipe

Original: {{ percentage }}
percent: {{ percentage | percent }}
percent:'1.0-2': {{ percentage | percent:'1.0-2' }}

📋 Array Pipes

SlicePipe

Original: {{ arrayOfStrings }}
slice:0:3: {{ arrayOfStrings | slice:0:3 }}
slice:1:4: {{ arrayOfStrings | slice:1:4 }}
slice:-3: {{ arrayOfStrings | slice:-3 }}

🔍 Object Pipes

JsonPipe

Original: {{ jsonObject }}
json: {{ jsonObject | json }}

⚡ Async Pipe

AsyncPipe

Promise result: {{ promiseValue | async }}
This will show "Async data loaded!" after 2 seconds

🎯 KeyValue Pipe

KeyValuePipe

Object entries:
  • {{ item.key }}: {{ item.value }}

💡 Tips for Using Pipes

  • Pipes are pure by default, meaning they only execute when their input changes
  • You can chain multiple pipes using the pipe operator (|)
  • Some pipes accept parameters separated by colons
  • The async pipe automatically handles subscription and unsubscription
  • Custom pipes can be created for specific transformation needs
