🔄 Simple Parent-Child Communication Demo

This demo shows the basic concepts of parent-child communication in Angular:

  • 📥 Parent → Child: Using @Input() properties
  • 📤 Child → Parent: Using @Output() events

👨‍💼 Parent Component

📥 Send Data to Child:

{{ parentMessage }}

{{ parentCount }}

📤 Data from Child:

Messages from Child:
{{ message }}
No messages from child yet...
Counts from Child:
{{ count }}
No counts from child yet...

💡 Code Examples

📥 Parent → Child (@Input)

// In Child Component
@Input() messageFromParent: string = '';
@Input() countFromParent: number = 0;

// In Parent Template
<app-child
  [messageFromParent]="parentMessage"
  [countFromParent]="parentCount">
</app-child>

📤 Child → Parent (@Output)

// In Child Component
@Output() messageToParent = new EventEmitter<string>();
@Output() countToParent = new EventEmitter<number>();

// In Parent Template
<app-child
  (messageToParent)="onMessageFromChild($event)"
  (countToParent)="onCountFromChild($event)">
</app-child>
