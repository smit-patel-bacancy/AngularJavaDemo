üìö Binding Summary

String Interpolation

  • {{ title }} - Display component data
  • Can include expressions, method calls, and operators
  • Automatically updates when data changes

Property Binding

  • [property]="value" - Bind component to DOM
  • One-way: Component ‚Üí DOM
  • Used for attributes, properties, and styles

Event Binding

  • (event)="handler()" - Listen to DOM events
  • One-way: DOM ‚Üí Component
  • Executes component methods on events

Two-Way Binding

  • [(ngModel)]="property" - Bidirectional binding
  • Combines property and event binding
  • Perfect for form inputs
