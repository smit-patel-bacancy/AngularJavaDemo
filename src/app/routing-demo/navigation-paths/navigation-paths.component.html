Understanding Navigation Paths

Learn about different types of navigation paths in Angular, including absolute paths, relative paths, and how to construct them properly.

Types of Navigation Paths

Absolute Paths

routerLink="/home"
routerLink="/products/123"
routerLink="/admin/users"

Start with forward slash (/) and navigate from the root of the application.

Example: /home, /products/123, /admin/users

Relative Paths

routerLink="./child"
routerLink="../parent"
routerLink="../../grandparent"

Navigate relative to the current route location.

Example: ./child, ../parent, ../../grandparent

Root-Relative Paths

routerLink="/"
routerLink="/dashboard"

Always navigate from the application root, regardless of current location.

Example: /, /dashboard, /profile

Path Construction Examples

Simple Navigation

// From any location
<a routerLink="/home">Home</a>
<a routerLink="/about">About</a>
<a routerLink="/contact">Contact</a>

Direct navigation to specific routes from anywhere in the app.

Parameterized Paths

// Dynamic parameters
<a [routerLink]="['/user', userId]">User Profile</a>
<a [routerLink]="['/product', productId, 'details']">Product Details</a>

Construct paths with dynamic parameters from component properties.

Nested Paths

// Child routes
<a routerLink="./details">Details</a>
<a routerLink="./settings">Settings</a>
<a routerLink="../list">Back to List</a>

Navigate within nested route structures using relative paths.

Advanced Path Techniques

Matrix Parameters

<a [routerLink]="['/products']"
   [queryParams]="{{ '{' }}category: 'electronics' {{ '}' }}">
   Electronics
</a>

Add query parameters to navigation paths.

Fragment Navigation

<a routerLink="/page" fragment="section1">
   Go to Section 1
</a>

Navigate to specific sections within a page.

Preserve Query Parameters

<a [routerLink]="['/search']"
   [queryParams]="{{ '{' }}query: searchTerm {{ '}' }}"
   [queryParamsHandling]="'merge'">
   Search
</a>

Preserve existing query parameters while adding new ones.

Conditional Navigation

<a [routerLink]="isAdmin ? '/admin' : '/user'">
   {{ '{' }} isAdmin ? 'Admin Panel' : 'User Dashboard' {{ '}' }}
</a>

Navigate based on component state or conditions.
