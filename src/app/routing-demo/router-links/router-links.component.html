Navigating with Router Links

Learn how to use Angular's routerLink directive to create navigation links that work seamlessly with the Angular Router.

Basic Router Links

Simple Navigation

<a routerLink="/home">Home</a>
<a routerLink="/about">About</a>
<a routerLink="/contact">Contact</a>

Basic navigation links that navigate to specific routes.

With Active State

<a routerLink="/home" routerLinkActive="active">Home</a>
<a routerLink="/about" routerLinkActive="active">About</a>
<a routerLink="/contact" routerLinkActive="active">Contact</a>

Links with active state styling when the route is active.

Multiple Active Classes

<a routerLink="/home"
   routerLinkActive="active selected">Home</a>

Apply multiple CSS classes when the link is active.

Advanced Router Link Features

Route Parameters

<a [routerLink]="['/user', userId]">User Profile</a>
<a routerLink="/product/123">Product Details</a>

Navigate to routes with dynamic parameters.

Query Parameters

<a [routerLink]="['/search']"
   [queryParams]="{{ '{' }}query: 'angular', page: 1 {{ '}' }}">
   Search Results
</a>

Add query parameters to the URL.

Fragment Navigation

<a routerLink="/page" fragment="section1">
   Go to Section 1
</a>

Navigate to specific sections within a page.

Relative Navigation

<a routerLink="./child">Child Route</a>
<a routerLink="../parent">Parent Route</a>

Navigate relative to the current route.

Router Link Active Options

Exact Match

<a routerLink="/home"
   routerLinkActive="active"
   [routerLinkActiveOptions]="{{ '{' }}exact: true {{ '}' }}">
   Home
</a>

Only apply active class when the route matches exactly.

Multiple Routes

<a routerLink="/admin"
   routerLinkActive="active"
   [routerLinkActiveOptions]="{{ '{' }}paths: ['admin', 'admin/users'] {{ '}' }}">
   Admin
</a>

Apply active class for multiple route patterns.

Button Navigation

Button with Router Link

<button [routerLink]="['/dashboard']">
   Go to Dashboard
</button>

Use routerLink on button elements for navigation.

Styled Navigation Button

<button class="nav-btn"
        [routerLink]="['/profile']"
        routerLinkActive="active">
   <i class="icon-user"></i>
   Profile
</button>

Create styled navigation buttons with icons.

Best Practices

✅ Use Semantic HTML

Use appropriate HTML elements for navigation (nav, ul, li).

✅ Provide Clear Labels

Use descriptive text that clearly indicates the destination.

✅ Handle Active States

Always use routerLinkActive to provide visual feedback.

✅ Consider Accessibility

Add proper ARIA labels and keyboard navigation support.

✅ Use Relative Paths

Prefer relative paths for better maintainability.

Common Navigation Patterns

Navigation Menu

<nav class="main-nav">
  <ul>
    <li><a routerLink="/home" routerLinkActive="active">Home</a></li>
    <li><a routerLink="/products" routerLinkActive="active">Products</a></li>
    <li><a routerLink="/about" routerLinkActive="active">About</a></li>
    <li><a routerLink="/contact" routerLinkActive="active">Contact</a></li>
  </ul>
</nav>

Breadcrumb Navigation

<nav class="breadcrumb">
  <a routerLink="/">Home</a>
  <span> / </span>
  <a routerLink="/products">Products</a>
  <span> / </span>
  <span>Current Page</span>
</nav>

Tab Navigation

<div class="tabs">
  <a routerLink="./details" routerLinkActive="active">Details</a>
  <a routerLink="./settings" routerLinkActive="active">Settings</a>
  <a routerLink="./history" routerLinkActive="active">History</a>
</div>
